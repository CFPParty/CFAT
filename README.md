# CFAT
Citizens' Fight Against Tookit 公民反抗工具包

## 一、项目用途
本项目可在公民反抗运动发生时启用，处理公民反抗运动的信息。

## 二、功能(对号为已实现，横线为待实现)(初期版本功能不全面，日后完善)
### 公民委员会端
- 公民运动参与者信息登记（个人信息登记）
- 公民纠察队信息登记（个人信息登记、任务分配、警情登记）
- 运动公告
- 暴力机关信息登记(信息登记、机关和人员处置任务分配)
### 运动参与者端
- 个人信息登记
### 公民纠察队端
- 警情上报
- 任务接收
- 执法记录
### 市民端
- 报警
- 了解运动
### 公共功能
- 文字对话IM
- 对讲机VOIP
